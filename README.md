# 🎯 That Tech Girl's Portfolio

> *"Because apparently 'eliminating bad code' is a more socially acceptable career than actual elimination."*

## 🕵️ What Is This?

Okay so like... this is my portfolio site but make it ✨dramatic✨. I got really into the whole cyberpunk/hitman aesthetic (don't judge me) and decided my web dev services needed to sound WAY cooler than they actually are.

Instead of "I build websites" it's now "TARGET ACQUISITION" and honestly? I'm not even sorry.

## 🎮 The Vibe

Think: freelance web developer meets dramatic noir film meets that one girl who spent too much time watching Mr. Robot. The whole site is dark, mysterious, and uses terms like "operator" and "contract" when really I'm just... making websites in my pajamas while eating cereal.

**Real translation guide:**
- "Target Acquisition" = I make new websites
- "Site Infiltration" = I fix existing websites
- "Silent Cleanup" = I optimize your site (remove the garbage code)
- "Weapon Upgrade" = I add new features
- "Identity Fabrication" = I do branding/design
- "Operator" = ...me. A girl with a laptop and too much coffee

## 🛠️ Tech Stack

Built with the actual good stuff:
- **React 18** - Because I'm not a monster
- **TypeScript** - Type safety keeps me sane
- **Vite** - Fast builds = happy developer
- **Tailwind CSS v4** - Utility classes go brrrr
- **React Router v7** - For that smooth SPA navigation
- **EmailJS** - So people can actually contact me
- **A TON of Radix UI** - Accessible components ftw
- **Motion** - For the fancy animations that make it look like I know what I'm doing

## 🚀 Getting Started

**Install stuff:**
```bash
pnpm install
```

**Run it locally:**
```bash
pnpm dev
```

**Build for production:**
```bash
pnpm build
```

**Deploy:**
Just push to `main` and Vercel handles it. Living in the future is cool.

## 📧 EmailJS Setup

The contact form uses EmailJS because I didn't want to deal with backend stuff. Set up your `.env` file with:

```bash
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

Check out `EMAILJS-SETUP.md` for the full setup guide (yes, I actually wrote docs, I know, shocking).

Listen, I know the whole "web developer as mysterious operative" thing is extra. But honestly? It makes filling out a contact form feel way more interesting than "Please tell me about your project uwu."
